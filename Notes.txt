7) current detection - acs770 (150 amp) -Done
8) ubec power supply - dzs elec mini 6-24v -> 5v 3A usb -Done
    
Software:
1) raspi-config
    a) enable ssh
    b) enable i2c
    c) disable serial terminal

2) apt-get install python-pip python-dev python-smbus git
3) git clone https://github.com/adafruit/Adafruit_Python_GPIO.git
4) cd Adafruit_Python_GPIO
5) sudo python setup.py install
6) git clone https://github.com/adafruit/Adafruit_Python_BME280.git
7) cd Adafruit_Python_BME280
9) sudo python setup.py install

11) cd Adafruit_Python_BNO055
